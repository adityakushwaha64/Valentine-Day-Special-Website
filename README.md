# Valentine Day Special Website ❤️

A creative and interactive Valentine Day themed website built using **HTML with embedded CSS and JavaScript**.

## ✨ Features
- Romantic UI design
- Interactive animations
- Embedded CSS styling
- Embedded JavaScript effects
- Responsive layout
- Single-file implementation

## 🛠️ Technologies Used
- HTML5
- CSS3 (embedded)
- JavaScript (embedded)

## 📂 Project Structure
```text
Valentine-Day-Special-Website/
│
├── index.html
└── README.md
```

## 🚀 How to Run
1. Download or clone the repository.
2. Open `index.html` in any web browser.
3. Enjoy the interactive Valentine Day website.

## 🎯 Purpose
This project was created to practice front-end web development, animations, and interactive design using a single HTML file.
